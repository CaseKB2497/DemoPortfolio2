Welcome to My Portfolio

Thank you for visiting my personal portfolio website. The purpose of this site is primarily to practice some of my more rusty front-end development skills. Additionally, I wanted to see how useful OpenAI's ChatGPT artificial intelligence was in the development process. I used the AI to generate templates for most of the pages seen here, using HTML, CSS, & Bootstrap. I figured this would be a neat way to showcase my skillset or lack thereof and highlight a few different coding projects I've worked on. Finally, I wanted a chance to use my newfound AWS knowledge to host the site on an EC2 instance.

Afterthought: As of right now, January 2023, this site has no accessibility features which is a crucial component of any modern technology. I am aiming to update this semi-frequently to improve the user experience, add accessibility options, and allow proper rescaling for mobile devices as I become more comfortable with front-end development.
